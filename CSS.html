Général

La majorité concerne le modèle de “boîtes” applicable sur n'importe quelle balise HTML
Nom Description Valeurs (exemple)
background-color Couleur d'arrière plan red
border-style Style a utiliser pour la bordure solid
border-width Taille de la bordure 1px
margin Taille de la marge extérieur a la bordure 5px
padding Taille de la marge intérieur a la bordure 5px

Texte

Utiliser pour formater du texte
Nom Description Valeurs (exemple)
font-family Police de charactère a utiliser 'Gill Sans', 'Gill Sans MT'
font-size Taille du texte 2em
font-style Définit si le texte est en italique italic
font-weight Définit si le texte est plus ou moins en gras bold
